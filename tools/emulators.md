# Sega Saturn Emulators and Tools

## Emulators
- Ymir: High-accuracy, with LLE CD, cache options, debugger. (This project's focus)
- Kronos: Fork of Yabause, accurate.
- Mednafen/Beetle: Good compatibility.
- SSF: Windows, high accuracy for some.
- Yabause: Older.

## Debugging/RE Tools
- VDP1/VDP2 viewers in emulators.
- SCU DSP debugger.
- Memory viewers, tracers.

## Homebrew
- Pseudo Saturn Kai: For backups, cheats.
- Saturn SDK samples.
- ChibiAkumas tutorials.
