# Anti-Piracy and Security

## Disc Format Protections (ST-040)
- Specific track layout, pauses, lead-in data.
- "Semi CD-ROM XA".
- 12cm only, timing restrictions.

## Physical Rings
- Outer edge pressed rings for authentication (visible anti-piracy feature).

## IP.BIN Security
- Specific code/object from SDK required for boot.
- See ipbin.md.

## Other
- Some games have additional checks.

## RE Notes
- Emulators often bypass or log security checks.
- Tools like Pseudo Saturn Kai for backups.

References: ST-040, Ymir/Kronos security handling, SegaXtreme threads.
